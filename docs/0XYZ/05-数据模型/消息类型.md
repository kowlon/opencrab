# 消息类型

**文件**: `src/seeagent/llm/types.py` 及相关

## 消息角色

```python
class MessageRole(Enum):
    """消息角色枚举"""
    
    SYSTEM = "system"           # 系统消息
    USER = "user"               # 用户消息
    ASSISTANT = "assistant"     # 助手消息
    TOOL = "tool"               # 工具结果消息
```

## 核心数据结构

### Message

```python
@dataclass
class Message:
    """对话消息"""
    
    role: str                           # system/user/assistant/tool
    content: str | list[ContentBlock]   # 消息内容
    name: str | None = None             # 名称（用于tool消息）
    tool_call_id: str | None = None     # 关联的工具调用ID
    metadata: dict = field(default_factory=dict)
```

### ContentBlock

```python
@dataclass
class ContentBlock:
    """内容块 - 用于复杂消息内容"""
    
    type: str                           # text/tool_use/tool_result
    text: str | None = None             # 文本内容
    id: str | None = None               # 工具调用ID (tool_use)
    name: str | None = None             # 工具名称 (tool_use)
    input: dict | None = None           # 工具参数 (tool_use)
    content: str | None = None          # 工具结果 (tool_result)
```

### ToolCall

```python
@dataclass
class ToolCall:
    """工具调用"""
    
    id: str                             # 调用ID
    name: str                           # 工具名称
    arguments: dict                     # 工具参数
```

### ToolUseBlock

```python
@dataclass
class ToolUseBlock(ContentBlock):
    """工具调用块"""
    
    type: str = "tool_use"
    id: str | None = None
    name: str | None = None
    input: dict | None = None
```

### ToolResultBlock

```python
@dataclass
class ToolResultBlock(ContentBlock):
    """工具结果块"""
    
    type: str = "tool_result"
    id: str | None = None
    content: str | None = None
```

## 消息示例

### 用户消息

```json
{
    "role": "user",
    "content": "帮我写一个Python函数计算斐波那契数列"
}
```

### 助手消息（带工具调用）

```json
{
    "role": "assistant",
    "content": [
        {
            "type": "text",
            "text": "我来帮你写一个斐波那契函数。"
        },
        {
            "type": "tool_use",
            "id": "toolu_123",
            "name": "write_file",
            "input": {
                "path": "fibonacci.py",
                "content": "def fib(n):\n    ..."
            }
        }
    ]
}
```

### 工具结果消息

```json
{
    "role": "tool",
    "content": "文件已成功写入: fibonacci.py",
    "tool_call_id": "toolu_123",
    "name": "write_file"
}
```

## Channel 消息格式

IM 渠道的消息格式：

```python
@dataclass
class ChannelMessage:
    """渠道消息"""
    
    id: str                             # 消息ID
    session_id: str                     # 会话ID
    content: str                        # 文本内容
    role: str = "user"                  # 角色
    channel: str                        # 渠道类型 (telegram/feishu/dingtalk...)
    user_id: str = ""                   # 用户ID
    chat_id: str = ""                   # 聊天ID
    timestamp: datetime                 # 时间戳
    attachments: list[Attachment] = field(default_factory=list)
    metadata: dict = field(default_factory=dict)
    reply_to: str | None = None         # 回复的消息ID
    mentions: list[str] = field(default_factory=list)  # @提及的用户
```
