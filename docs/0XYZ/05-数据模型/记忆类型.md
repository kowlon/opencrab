# 记忆类型

**文件**: `src/seeagent/memory/types.py`

## 记忆类型枚举

```python
class MemoryType(Enum):
    """记忆类型枚举"""
    
    SEMANTIC = "semantic"        # 实体-属性结构化记忆
    EPISODE = "episode"          # 完整交互故事
    SCRATCHPAD = "scratchpad"    # 跨会话工作记忆
```

## 核心数据结构

### Memory

```python
@dataclass
class Memory:
    """通用记忆基类"""
    
    id: str                           # 唯一标识符
    content: str                      # 记忆内容
    memory_type: MemoryType           # 记忆类型
    created_at: datetime              # 创建时间
    updated_at: datetime | None = None  # 更新时间
    scope: str = "global"             # 作用域 (global/org)
    scope_owner: str = ""             # 作用域所有者
    tags: list[str] = field(default_factory=list)  # 标签
    metadata: dict = field(default_factory=dict)   # 额外元数据
    source: str = ""                  # 来源 (user/input/agent/extracted)
    confidence: float = 1.0           # 置信度 (0-1)
    embedding: list[float] | None = None  # 向量表示
```

### SemanticMemory

```python
@dataclass
class SemanticMemory(Memory):
    """语义记忆 - 实体-属性结构"""
    
    memory_type: MemoryType = MemoryType.SEMANTIC
    
    entity: str = ""                  # 实体名称
    attributes: dict = field(default_factory=dict)  # 属性键值对
    relations: list[str] = field(default_factory=list)  # 关联实体
    category: str = ""                # 类别
```

### Episode

```python
@dataclass
class Episode:
    """情节记忆 - 完整交互故事"""
    
    id: str
    session_id: str                   # 关联的会话ID
    start_time: datetime
    end_time: datetime | None = None
    task_description: str             # 任务描述
    summary: str = ""                 # 摘要
    turns: list[Turn] = field(default_factory=list)  # 对话轮次
    outcome: str = ""                 # 结果 (success/failure)
    errors: list[str] = field(default_factory=list)  # 错误列表
    lessons: list[str] = field(default_factory=list)  # 学到的教训

@dataclass
class Turn:
    """对话轮次"""
    
    role: str                         # user/assistant/system
    content: str
    tool_calls: list[dict] = field(default_factory=list)
    tool_results: list[dict] = field(default_factory=list)
    timestamp: datetime
```

### Scratchpad

```python
@dataclass
class Scratchpad:
    """工作记忆 - 跨会话持久化"""
    
    id: str
    content: str                      # 工作笔记内容
    created_at: datetime
    updated_at: datetime
    session_id: str | None = None     # 创建时所属会话
    scope: str = "global"
    scope_owner: str = ""
    is_active: bool = True            # 是否活跃
    expires_at: datetime | None = None  # 过期时间
```

### Attachment

```python
@dataclass
class Attachment:
    """附件 - 文件/媒体"""
    
    id: str
    filename: str                     # 文件名
    mime_type: str                    # MIME类型
    file_size: int = 0                # 文件大小（字节）
    local_path: str = ""              # 本地路径
    url: str = ""                     # 远程URL
    description: str = ""             # 描述
    transcription: str = ""           # 语音/视频转录
    extracted_text: str = ""          # 提取的文本
    tags: list[str] = field(default_factory=list)
    direction: str = "inbound"        # inbound/outbound
    session_id: str
    created_at: datetime
    original_filename: str = ""       # 原始文件名
```

## 存储格式

### SQLite Schema

```sql
CREATE TABLE memories (
    id TEXT PRIMARY KEY,
    content TEXT NOT NULL,
    memory_type TEXT NOT NULL,
    scope TEXT DEFAULT 'global',
    scope_owner TEXT DEFAULT '',
    tags TEXT,  -- JSON array
    metadata TEXT,  -- JSON object
    source TEXT DEFAULT '',
    confidence REAL DEFAULT 1.0,
    embedding BLOB,  -- for vector search backend
    created_at TEXT NOT NULL,
    updated_at TEXT
);

CREATE VIRTUAL TABLE memories_fts USING fts5(
    content,
    tags,
    content='memories',
    content_rowid='rowid'
);
```

### JSONL 格式

用于 Episode 存储：

```jsonl
{"id": "ep_xxx", "session_id": "sess_xxx", "task": "...", "summary": "...", "outcome": "success"}
```
