# 工具定义

**文件**: `src/seeagent/tools/definitions/` 及 `tools/handlers/*.py`

## Tool 定义

```python
@dataclass
class Tool:
    """工具定义"""
    
    name: str                           # 工具名称
    description: str                    # 工具描述（供LLM理解）
    input_schema: dict                  # 输入参数JSON Schema
    output_schema: dict | None = None   # 输出JSON Schema
    handler: ToolHandler | None = None  # 处理器实例
    tags: list[str] = field(default_factory=list)
    enabled: bool = True
    metadata: dict = field(default_factory=dict)
```

## ToolHandler 基类

```python
class ToolHandler(ABC):
    """工具处理器基类"""
    
    @property
    @abstractmethod
    def name(self) -> str:
        """工具名称"""
    
    @property
    def description(self) -> str:
        """工具描述"""
        return ""
    
    @property
    def input_schema(self) -> dict:
        """输入JSON Schema"""
        return {"type": "object", "properties": {}}
    
    @property
    def output_schema(self) -> dict | None:
        """输出JSON Schema"""
        return None
    
    @abstractmethod
    async def execute(self, **kwargs) -> Any:
        """执行工具"""
        pass
    
    async def validate(self, arguments: dict) -> bool:
        """验证参数"""
        return True
    
    async def on_error(self, error: Exception, **kwargs) -> str:
        """错误处理"""
        return f"Error: {str(error)}"
```
