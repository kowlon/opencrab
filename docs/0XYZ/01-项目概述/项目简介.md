# 项目简介

## 项目背景

OpenCrab（SeeAgent）是一个**多智能体 AI 助手系统**，其核心理念是"Ralph Wiggum 模式"——一种永不放弃的执行模式。通过状态持久化和智能重试机制，系统能够从失败中恢复并继续执行复杂任务。

> "Ralph Wiggum 模式"源自辛普森一家中的 Ralph Wiggum 角色——他纯真、善良，永远不会放弃。

## 核心特性

### 1. 多智能体协作

- 支持创建多个专业化的 Agent（内容创作、代码助手、数据分析等）
- 支持任务委派（最大深度 5 层）
- 子 Agent 共享完整会话上下文

### 2. 三层记忆系统

| 层次 | 类型 | 说明 |
|------|------|------|
| 第一层 | SemanticMemory | 实体-属性结构化记忆 |
| 第二层 | Episode | 完整交互故事/情节 |
| 第三层 | Scratchpad | 跨会话持久化工作记忆 |

### 3. 渐进式暴露

工具和技能按需暴露给 LLM，避免上下文膨胀。

### 4. 自我进化

- 自动分析失败模式
- 动态创建和安装新技能
- 从日志中学习

## 技术栈

### 后端

- **语言**: Python 3.11+
- **框架**: FastAPI、asyncio
- **数据库**: aiosqlite（SQLite 异步封装）
- **向量检索**: ChromaDB（可选）

### 前端

- **桌面**: Tauri 2.x + React 18
- **Web**: React 18 + TypeScript + Vite 6

### LLM 支持

- Anthropic Claude 系列
- OpenAI 兼容 API（30+ 提供商）

### IM 渠道

- Telegram、飞书、钉钉、企业微信、QQ、OneBot
