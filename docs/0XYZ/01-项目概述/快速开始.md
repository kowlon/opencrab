# 快速开始

## 环境准备

### Python 环境

```bash
# Python 3.11+ 推荐
python --version

# 创建虚拟环境
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate

# 安装项目
pip install -e ".[dev]"
```

### 前端环境（仅需修改 frontend 时）

```bash
cd apps/setup-center && npm install
```

## 运行方式

### CLI 交互模式

```bash
seeagent
```

### 单次任务模式

```bash
seeagent run "你的任务描述"
```

### API 服务模式

```bash
seeagent serve
```

服务启动后可访问：
- API 文档: `http://localhost:18900/docs`
- 健康检查: `http://localhost:18900/health`

### 桌面应用模式

```bash
cd apps/setup-center && npm run tauri dev
```

## 配置

### LLM 配置

在 `data/config.yaml` 或环境变量中配置 API Key：

```yaml
llm:
  provider: anthropic
  api_key: your-api-key
  model: claude-sonnet-4-20250514
```

### IM 渠道配置

各渠道配置见 [渠道集成](../06-使用指南/渠道集成.md)。

## 测试

```bash
# 运行所有测试
pytest

# 运行单元测试
pytest tests/unit/

# 运行特定测试
pytest -k "test_brain"

# 带覆盖率
pytest --cov=src/seeagent
```

## 代码质量

```bash
# Lint
ruff check src/

# 格式化
ruff format src/

# 类型检查
mypy src/seeagent/
```
