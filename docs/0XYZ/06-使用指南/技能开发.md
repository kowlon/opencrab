# 技能开发

## 技能格式

技能使用 `SKILL.md` 声明式定义。

## 目录结构

```
skills/
└── my-skill/
    ├── SKILL.md           # 必须：技能定义
    ├── handler.py         # 可选：自定义处理器
    └── README.md          # 可选：文档
```

## SKILL.md 结构

```markdown
# Skill: my-skill

## Description
这是一个示例技能...

## Triggers
- 用户请求执行任务
- 关键词触发

## Actions
1. 第一步操作
2. 第二步操作

## Parameters
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| param1 | string | 是 | 参数1 |
| param2 | number | 否 | 参数2 |

## Output
返回执行结果描述...

## Examples
用户: "帮我执行任务"
Agent: 执行结果...
```

## 示例技能

### hello-world

```markdown
# Skill: hello-world

## Description
输出欢迎信息

## Triggers
- 用户说"你好"
- 用户说"hello"

## Actions
1. 输出问候语

## Output
"你好！我是 SeeAgent，很高兴为您服务！"
```

### file-processor

```markdown
# Skill: file-processor

## Description
处理文件内容

## Triggers
- 用户请求处理文件
- 关键词: "处理文件"、"分析文件"

## Actions
1. 使用 read_file 读取文件
2. 分析文件内容
3. 返回结果

## Parameters
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| path | string | 是 | 文件路径 |
| operation | string | 否 | 操作类型(read/analyze) |

## Output
处理后的结果...
```

## 安装技能

### 从本地安装

```bash
seeagent skill install ./skills/my-skill
```

### 从 Git 安装

```bash
seeagent skill install https://github.com/user/repo --subdir skills/my-skill
```

### 从 SKILL.md URL 安装

```bash
seeagent skill install https://example.com/SKILL.md
```

## 查看已安装技能

```bash
seeagent skill list
```

## 技能加载顺序

1. `__builtin__` — 内置技能
2. `__user_workspace__` — 工作区技能
3. `skills/` — 项目技能
4. `.cursor/skills` — Cursor IDE
5. `.claude/skills` — Claude IDE
6. 全局 home 目录
