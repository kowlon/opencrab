# 渠道集成

## 支持的渠道

| 渠道 | 协议 | 配置文件 |
|------|------|----------|
| Telegram | Telegram Bot API | `data/channels/telegram.yaml` |
| Feishu | 飞书开放平台 | `data/channels/feishu.yaml` |
| DingTalk | 钉钉开放平台 | `data/channels/dingtalk.yaml` |
| WeCom | 企业微信 | `data/channels/wework.yaml` |
| OneBot | OneBot 协议 | `data/channels/onebot.yaml` |
| QQ | QQ 官方 Bot | `data/channels/qq.yaml` |

## 通用配置

```yaml
channels:
  enabled: true
  gateway:
    port: 8080
    host: 0.0.0.0
```

## Telegram 配置

```yaml
channels:
  telegram:
    enabled: true
    bot_token: "123456:ABC-..."
    allowed_users:
      - user_id_1
      - user_id_2
    group_mode: "first"  # first/only/all
    admin_users:
      - admin_user_id
```

**设置步骤**:
1. 在 Telegram BotFather 创建机器人，获取 token
2. 配置文件填入 token
3. 重启服务
4. 给机器人发送 `/start`

## 飞书配置

```yaml
channels:
  feishu:
    enabled: true
    app_id: "cli_xxx"
    app_secret: "xxx"
    bot_name: "SeeAgent"
```

**设置步骤**:
1. 在飞书开放平台创建企业自建应用
2. 配置机器人能力
3. 获取 App ID 和 App Secret
4. 配置事件订阅（需要公网地址）

## 钉钉配置

```yaml
channels:
  dingtalk:
    enabled: true
    app_key: "dingxxx"
    app_secret: "xxx"
    robot_code: "robot_code"
```

## 企业微信配置

### HTTP 回调模式

```yaml
channels:
  wework:
    enabled: true
    corp_id: "wwxxx"
    agent_id: "1000001"
    secret: "xxx"
    token: "xxx"
    encoding_aes_key: "xxx"
    callback_mode: "http"
```

### WebSocket 模式

```yaml
channels:
  wework:
    enabled: true
    corp_id: "wwxxx"
    agent_id: "1000001"
    secret: "xxx"
    callback_mode: "websocket"
```

## OneBot 配置

```yaml
channels:
  onebot:
    enabled: true
    adapter: "napcat"  # napcat/lagrange
    ws_url: "ws://localhost:3001"
    token: "optional_token"
```

## 通用命令

| 命令 | 说明 |
|------|------|
| `/start` | 开始对话 |
| `/stop` | 停止当前任务 |
| `/help` | 显示帮助 |
| `/status` | 显示状态 |
| `/reset` | 重置会话 |
