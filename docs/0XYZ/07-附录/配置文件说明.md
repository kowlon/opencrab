# 配置文件说明

## 配置文件位置

| 文件 | 说明 |
|------|------|
| `data/config.yaml` | 主配置 |
| `data/profiles/*.json` | Agent 配置 |
| `data/channels/*.yaml` | 渠道配置 |
| `identity/runtime/*.json` | 编译后身份片段 |

## 主配置 (config.yaml)

```yaml
# LLM 配置
llm:
  provider: anthropic        # anthropic / openai
  api_key: ${ANTHROPIC_API_KEY}  # 支持环境变量
  model: claude-sonnet-4-20250514
  max_tokens: 4096
  temperature: 1.0

# 记忆配置
memory:
  search_backend: fts5       # fts5 / chromadb / api
  embedding_model: ""
  embedding_device: cpu      # cpu / cuda
  vector_dimension: 384

# 调度配置
scheduler:
  enabled: true
  max_concurrent: 5
  timezone: Asia/Shanghai

# 日志配置
logging:
  level: INFO               # DEBUG / INFO / WARNING / ERROR
  format: "%(asctime)s - %(name)s - %(levelname)s - %(message)s"
  file: data/logs/seeagent.log

# 服务器配置
server:
  host: 0.0.0.0
  port: 8000
  cors_origins:
    - "*"
```
