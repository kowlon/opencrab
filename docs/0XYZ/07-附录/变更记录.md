# 变更记录

## v1.27.0 (开发中)

### 新增功能
- Brain 层流式化，支持真正的 idle timeout 语义
- ask_user prompt 防照抄机制
- MemoryConsolidator 支持时间戳前缀（`{YYYYMMDDHHMMSS}__{session_id}.jsonl`）和文件迁移
- 多轮对话上下文提升最佳实践匹配与参数提取准确率
- SSE 周期心跳防止 LB/CDN 空闲断连

### 改进
- ReasoningEngine 检查点机制优化
- bp_ask_user 提问内容质量提升
- LLM timeout 阈值优化（8K-60K tokens 中等请求不再因阈值过高而超时）

### Bug 修复
- 修复 camera-frame-search 字段提取分支漂移
- 修复 run_skill_script 路径 bug
- 修复 camera-frame-search config.yaml 嵌套双引号导致 YAML 解析失败
- 修复用户消息未走 BP 流程的三层问题
