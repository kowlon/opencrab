# 设计模式

## 1. Ralph Wiggum 模式

### 概念

永不放弃的执行模式，确保复杂任务能够完成。

### 实现

**文件**: `src/seeagent/core/ralph.py`

### 核心原理

```python
class RalphLoop:
    def run(self, task, max_retries=None):
        state = self.load_state()  # 尝试加载已保存状态
        while True:
            try:
                if state:
                    result = self.resume(state)  # 从断点恢复
                else:
                    result = self.execute(task)  # 全新执行
                self.save_success(result)
                return result
            except Failure as e:
                state = self.save_checkpoint(e)  # 保存断点
                if max_retries and self.retry_count > max_retries:
                    raise
                # 智能等待后重试
                wait_time = self.calculate_backoff(e)
                await asyncio.sleep(wait_time)
```

### 关键特性

| 特性 | 说明 |
|------|------|
| 状态持久化 | 将执行状态保存到文件系统 |
| 断点恢复 | 从失败点继续执行，而非从头开始 |
| 智能退避 | 根据失败类型计算重试间隔 |
| 永不放弃 | 默认无限重试（可配置上限） |

## 2. ReAct 推理模式

### 概念

显式的 Reason → Act → Observe 循环。

### 实现

**文件**: `src/seeagent/core/reasoning_engine.py`

### 核心原理

```python
class ReasoningEngine:
    async def think(self, task: str) -> str:
        thoughts = []
        history = []
        
        while not self.is_complete(task, history):
            # 1. Reason - 推理下一步行动
            reasoning = await self.reason(task, history)
            thoughts.append(reasoning)
            
            # 2. Act - 执行行动
            action = reasoning.action
            result = await self.execute(action)
            history.append((action, result))
            
            # 3. Observe - 观察结果
            observation = self.extract_observation(result)
            history.append(observation)
        
        return self.generate_final_answer(thoughts, history)
```

## 3. 多智能体委派模式

### 概念

通过 Agent 委派实现专业化分工。

### 实现

**文件**: `src/seeagent/agents/orchestrator.py`

### 核心原理

```python
class AgentOrchestrator:
    async def delegate(self, task: str, depth: int = 0) -> str:
        if depth >= MAX_DELEGATION_DEPTH:  # 最大深度 = 5
            raise MaxDepthExceeded()
        
        # 分析任务，决定是否委派
        sub_agent_type = self.analyze_task(task)
        if sub_agent_type:
            # 创建子 Agent
            agent = self.factory.create(sub_agent_type)
            result = await agent.run(task)
            return self.merge_results(result)
        else:
            # 自己执行
            return await self.agent.run(task)
```

## 4. 三层记忆模式

### 概念

语义记忆 + 情节记忆 + 工作记忆。

### 实现

**文件**: `src/seeagent/memory/manager.py`

### 核心原理

```python
class MemoryManager:
    async def store(self, content: str, memory_type: MemoryType):
        if memory_type == SEMANTIC:
            # 结构化实体-属性
            entity = self.extract_entity(content)
            await self.unified_store.put(entity)
            
        elif memory_type == EPISODE:
            # 完整交互故事
            episode = self.create_episode(content)
            await self.unified_store.put(episode)
            
        elif memory_type == SCRATCHPAD:
            # 跨会话工作记忆
            await self.unified_store.put_scratchpad(content)
    
    async def retrieve(self, query: str, limit: int = 10):
        # 多路径检索 + 重排序
        results = await self.retrieval.multi_path(query)
        return self.rerank(results, limit)
```

## 5. 渐进式暴露模式

### 概念

工具和技能按需暴露给 LLM，避免上下文膨胀。

### 实现

**文件**: `src/seeagent/tools/catalog.py`

### 核心原理

```python
class ToolCatalog:
    def __init__(self):
        self.high_frequency = [...]  # 高频工具（常驻）
        self.low_frequency = [...]   # 低频工具（按需）
        self.progressive_level = 0
    
    def get_tools_for_llm(self, context: Context) -> list[Tool]:
        tools = self.high_frequency.copy()
        
        # 根据上下文逐步添加更多工具
        if context.complexity > SIMPLE:
            tools.extend(self.medium_frequency[:self.progressive_level])
        if context.complexity > MEDIUM:
            tools.extend(self.low_frequency[:self.progressive_level])
        
        return tools
```

## 6. 工厂模式

### 概念

通过工厂创建差异化的 Agent 实例。

### 实现

**文件**: `src/seeagent/agents/factory.py`

### 核心原理

```python
class AgentFactory:
    def create(self, profile: AgentProfile) -> Agent:
        agent = Agent(
            brain=self.create_brain(profile),
            reasoning_engine=self.create_reasoning_engine(profile),
            tool_executor=self.create_tool_executor(profile),
            memory_manager=self.memory_manager,
        )
        return agent
    
    def create_brain(self, profile: AgentProfile) -> Brain:
        if profile.type == CODE_ASSISTANT:
            return CodeFocusedBrain(...)
        elif profile.type == CONTENT_CREATOR:
            return CreativeBrain(...)
        else:
            return GeneralBrain(...)
```

## 7. 观察者模式

### 概念

任务执行状态的实时通知。

### 实现

**文件**: `src/seeagent/core/task_monitor.py`

### 核心原理

```python
class TaskMonitor:
    def __init__(self):
        self.observers: list[Observer] = []
    
    def add_observer(self, observer: Observer):
        self.observers.append(observer)
    
    def notify(self, event: TaskEvent):
        for observer in self.observers:
            await observer.on_task_event(event)
```
