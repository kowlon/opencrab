# 数据流设计

## 1. 用户请求数据流

```
用户输入
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      MessageGateway                              │
│  channels/gateway.py                                            │
│                                                                  │
│  1. 协议转换（IM私有协议 → 内部Message格式）                      │
│  2. 会话关联（session_id）                                       │
│  3. 中断检测（CTRL+C）                                           │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                    AgentOrchestrator                             │
│  agents/orchestrator.py                                         │
│                                                                  │
│  1. 根据 session_id 获取 AgentInstance                          │
│  2. 判断是否需要委派给子 Agent                                   │
│  3. 路由消息                                                     │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      ReasoningEngine                             │
│  core/reasoning_engine.py                                       │
│                                                                  │
│  ReAct 循环:                                                     │
│  ┌────────────────────────────────────────────────────────────┐ │
│  │  Reason: LLM 推理下一步行动                                 │ │
│  │  Act: 执行工具或 LLM 调用                                   │ │
│  │  Observe: 观察结果，进入下一轮                              │ │
│  └────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
    │
    ├──► Brain (LLM 调用)
    │
    ├──► ToolExecutor (工具调用)
    │
    └──► MemoryManager (记忆读写)
```

## 2. LLM 调用数据流

```
PromptAssembly
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      PromptBuilder                               │
│  prompt/builder.py                                              │
│                                                                  │
│  组装顺序:                                                       │
│  1. Identity (SOUL.md 原则)                                     │
│  2. Persona                                                     │
│  3. Runtime                                                     │
│  4. Session Rules                                               │
│  5. AGENTS.md                                                   │
│  6. Catalogs (Tools/Skills/MCP)                                │
│  7. Memory (语义检索)                                           │
│  8. User                                                        │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                        Brain                                     │
│  core/brain.py                                                  │
│                                                                  │
│  1. 消息序列 → LLMRequest                                        │
│  2. 流式/非流式 调用                                             │
│  3. 工具调用 解析                                                │
│  4. 重试处理                                                    │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      LLMClient                                   │
│  llm/client.py                                                  │
│                                                                  │
│  1. Provider 选择                                               │
│  2. 请求格式化                                                  │
│  3. 响应解析                                                    │
│  4. Fallback 处理                                               │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      Provider                                    │
│  llm/providers/anthropic.py / openai.py                         │
│                                                                  │
│  API 调用 + 错误处理                                            │
└─────────────────────────────────────────────────────────────────┘
```

## 3. 工具执行数据流

```
ToolCall (from LLM)
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                     ToolExecutor                                 │
│  core/tool_executor.py                                          │
│                                                                  │
│  1. 工具名称 解析                                               │
│  2. 参数 验证                                                  │
│  3. 超时 设置                                                   │
│  4. 执行 调度                                                   │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      ToolCatalog                                 │
│  tools/catalog.py                                               │
│                                                                  │
│  1. 工具 查找                                                   │
│  2. 定义 获取                                                   │
│  3. Schema 验证                                                │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                     ToolHandler                                  │
│  tools/handlers/*.py                                            │
│                                                                  │
│  实际执行:                                                       │
│  - browser.py: Playwright 浏览器                                │
│  - filesystem.py: 文件读写                                      │
│  - shell.py: 命令执行                                           │
│  - web_search.py: 网页搜索                                      │
│  - ...                                                          │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
ToolResult
    │
    ▼
MemoryManager (可选存储)
    │
    ▼
返回 Brain 继续推理
```

## 4. 记忆系统数据流

### 4.1 写入流程

```
Interaction (对话/执行)
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                    MemoryExtractor                               │
│  memory/extractor.py                                            │
│                                                                  │
│  AI 提取关键信息:                                               │
│  - 实体/属性                                                    │
│  - 交互摘要                                                     │
│  - 工作笔记                                                     │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                   MemoryConsolidator                             │
│  memory/consolidator.py                                         │
│                                                                  │
│  1. 去重                                                        │
│  2. 合并相似记忆                                                │
│  3. 更新索引                                                    │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                    UnifiedStore                                  │
│  memory/unified_store.py                                        │
│                                                                  │
│  SQLite (FTS5)  ◄──────►  VectorStore (ChromaDB/API)           │
│     结构化数据              语义向量                              │
└─────────────────────────────────────────────────────────────────┘
```

### 4.2 读取流程

```
Query (检索查询)
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      Retrieval                                   │
│  memory/retrieval.py                                            │
│                                                                  │
│  多路径检索:                                                     │
│  1. 关键词 搜索 (FTS5)                                         │
│  2. 语义 搜索 (Vector)                                          │
│  3. 混合 搜索                                                   │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      ReRanking                                   │
│  memory/retrieval.py                                            │
│                                                                  │
│  AI 重排序:                                                     │
│  - 相关性 评分                                                  │
│  - 时效性 加权                                                  │
│  - 重要性 加权                                                  │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
Top-K Results
    │
    ▼
注入 Prompt
```

## 5. 会话生命周期数据流

```
创建会话
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                    SessionManager                                │
│  sessions/manager.py                                            │
│                                                                  │
│  1. 生成 session_id                                             │
│  2. 创建 Session 对象                                           │
│  3. 关联 User                                                   │
│  4. 初始化 AgentInstancePool                                    │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
消息处理循环
    │
    ├──► 消息 → AgentOrchestrator → Agent → ReasoningEngine
    │                                    │
    │                                    ▼
    │                              MemoryManager (读写)
    │
    ▼
会话关闭/超时
    │
    ▼
清理资源:
- AgentInstancePool 释放
- 临时文件清理
- 状态持久化
```

## 6. 技能加载数据流

```
系统启动 / 运行时发现
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      SkillLoader                                 │
│  skills/loader.py                                               │
│                                                                  │
│  加载顺序:                                                       │
│  1. __builtin__                                                │
│  2. __user_workspace__                                         │
│  3. skills/                                                    │
│  4. .cursor/skills                                             │
│  5. .claude/skills                                             │
│  6. Global home dirs                                           │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                      SkillParser                                 │
│  skills/parser.py                                              │
│                                                                  │
│  解析 SKILL.md:                                                │
│  - name, description                                           │
│  - triggers, actions                                           │
│  - parameters, returns                                          │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────┐
│                     SkillRegistry                                │
│  skills/registry.py                                            │
│                                                                  │
│  1. 注册技能                                                    │
│  2. 渐进式暴露 控制                                            │
│  3. 技能目录 生成                                              │
└─────────────────────────────────────────────────────────────────┘
    │
    ▼
ToolCatalog / Prompt 注入
```
