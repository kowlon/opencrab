# 模块关系图

## 模块依赖树

```
src/seeagent/
│
├── api/routes/                  # API 层（依赖所有业务模块）
│   ├── chat.py          ───────► agents/, memory/, sessions/
│   ├── agents.py        ───────► agents/, profiles/
│   ├── skills.py        ───────► skills/
│   └── memory.py        ───────► memory/
│
├── agents/                      # 多智能体层（依赖 core、sessions）
│   ├── orchestrator.py ───────► core/agent.py, sessions/
│   ├── factory.py       ───────► core/agent.py, profiles/
│   └── task_queue.py    ───────► sessions/
│
├── core/                        # 核心层（被所有业务模块依赖）
│   ├── agent.py         ───────► brain, reasoning_engine, tool_executor,
│   │                             memory/, skills/, sessions/
│   ├── brain.py         ───────► llm/client.py
│   ├── reasoning_engine.py ────► brain, tool_executor, memory/
│   ├── ralph.py         ───────► agent_state, memory/
│   ├── tool_executor.py ───────► tools/catalog.py
│   └── identity.py      ───────► prompt/compiler.py
│
├── tools/                       # 工具层（被 core、执行层依赖）
│   ├── catalog.py      ───────► definitions/, handlers/
│   ├── mcp.py          ───────► mcp_servers/
│   └── handlers/               # 具体工具实现
│       ├── browser.py  ───────► playwright, browser-use
│       ├── filesystem.py ─────► aiofiles
│       └── shell.py    ───────► asyncio
│
├── memory/                      # 记忆层（被 core、执行层依赖）
│   ├── manager.py      ───────► unified_store, retrieval, extractor
│   ├── unified_store.py ──────► storage, vector_store, search_backends
│   ├── retrieval.py    ───────► storage
│   └── search_backends.py ────► storage (FTS5, ChromaDB, API)
│
├── prompt/                      # 提示词层（被 core/identity 依赖）
│   ├── compiler.py     ───────► identity/
│   ├── builder.py      ───────► compiler, memory/retrieval
│   └── budget.py       ───────► token_tracking
│
├── llm/                         # LLM 层（被 core/brain 依赖）
│   ├── client.py       ───────► providers/
│   └── providers/
│       ├── anthropic.py
│       └── openai.py
│
├── sessions/                    # 会话层（被 api、agents 依赖）
│   ├── manager.py      ───────► session.py, user.py
│   └── session.py
│
├── channels/                    # 渠道层（独立入口）
│   └── gateway.py      ───────► sessions/, agents/
│
├── skills/                      # 技能层（被 core 依赖）
│   ├── loader.py       ───────► registry.py, catalog.py
│   ├── registry.py
│   └── parser.py
│
├── scheduler/                   # 调度层（独立服务）
│   ├── scheduler.py    ───────► executor.py
│   └── executor.py     ───────► core/agent.py
│
└── orgs/                        # 组织层（多租户）
    ├── manager.py      ───────► runtime.py, blackboard.py
    └── runtime.py      ───────► agent/, memory/
```

## 关键依赖关系

### Agent 核心依赖

```
Agent 依赖关系（由深到浅）:

Agent (core/agent.py)
├── ReasoningEngine (core/reasoning_engine.py)
│   ├── Brain (core/brain.py)
│   │   └── LLMClient (llm/client.py)
│   │       └── Providers (anthropic, openai)
│   ├── ToolExecutor (core/tool_executor.py)
│   │   └── ToolCatalog (tools/catalog.py)
│   │       └── ToolHandlers (browser, filesystem, shell...)
│   └── MemoryManager (memory/manager.py)
│       ├── UnifiedStore (memory/unified_store.py)
│       │   ├── Storage (SQLite + FTS5)
│       │   └── VectorStore (ChromaDB/API)
│       └── Retrieval (memory/retrieval.py)
├── Identity (core/identity.py)
│   └── PromptCompiler (prompt/compiler.py)
│       └── PromptBuilder (prompt/builder.py)
│           ├── MemoryRetriever (memory/retrieval.py)
│           └── ToolCatalog (tools/catalog.py)
└── SkillManager (core/skill_manager.py)
    └── SkillLoader (skills/loader.py)
```

### API 请求处理流程

```
HTTP Request
    │
    ▼
api/server.py (FastAPI)
    │
    ▼
api/routes/*.py (端点)
    │
    ├──► agents/orchestrator.py ──► AgentFactory ──► core/agent.py
    │
    ├──► memory/manager.py ──► memory/unified_store.py
    │
    ├──► skills/loader.py ──► skills/registry.py
    │
    └──► sessions/manager.py
```
