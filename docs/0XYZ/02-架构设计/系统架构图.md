## 层级说明

### 用户层 (User Layer)
用户通过多种方式与系统交互：
- CLI 交互界面
- REST API
- IM 渠道机器人（Telegram、飞书、钉钉等）

### 网关层 (Gateway Layer)
**MessageGateway** 统一处理所有输入：
- 协议转换（IM 私有协议 ↔ 内部格式）
- 会话关联
- 中断处理

### 协调层 (Orchestration Layer)
**AgentOrchestrator** 负责任务分发：
- 根据 Agent Profile 路由消息
- 管理多 Agent 协作
- 控制委派深度

### 执行层 (Execution Layer)
**Agent** 实例执行具体任务：
- **ReasoningEngine**: ReAct 推理循环
- **Brain**: LLM 交互
- **ToolExecutor**: 工具调用
- **MemoryManager**: 记忆管理

### 支撑层 (Supporting Layer)
提供基础设施支持：
- **LLMClient**: 多 Provider 支持
- **Skills**: 技能系统
- **Scheduler**: 定时任务
